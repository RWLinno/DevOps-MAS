# DevOps-MAS Demo

React + TypeScript + Vite + shadcn/ui 构建的智能运维问答系统交互式 Demo。

## 技术栈

- React 18 + TypeScript
- Vite 构建工具
- Tailwind CSS + shadcn/ui 组件库
- Recharts 可视化

## 开发

```bash
cd demo
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建产物在 `dist/` 目录，由 `server_unified.py` 在 `/demo` 路径下提供服务。
